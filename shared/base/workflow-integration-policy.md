# Workflow integration policy

The installed `shared/workflow-contract.md` is the authoritative optional `.ai-workflow/` protocol. Workflow use is opt-in and must never block standalone use. Read only relevant concise handoffs as unverified leads, re-verify material claims in repository evidence, preserve other state runs and unknown metadata, and write artifact and handoff before marking the current run complete.
