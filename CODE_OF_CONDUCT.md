# Code of Conduct

This project is intended to be welcoming and respectful.

## Expected behavior

- Be constructive.
- Be specific and factual.
- Respect differing approaches and implementation styles.
- Focus criticism on the work, not the person.

## Unacceptable behavior

- Harassment or intimidation
- Discriminatory language or behavior
- Personal attacks
- Deliberate disruption of collaboration

## Enforcement

Maintainers may remove content, close issues, or reject contributions that violate this code of conduct.

## Reporting

Report concerns to the repository maintainer through the normal project contact or issue channel.
