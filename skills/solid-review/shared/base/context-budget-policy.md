# Context budget policy

Load only the request, `SKILL.md`, and artifacts that can change the next decision; never preload references. Prioritize high-risk paths and state coverage/unknowns. Telemetry may retain artifact paths and purpose, never raw user, repository, secret, or artifact content. If blocked, name the next smallest evidence needed.
