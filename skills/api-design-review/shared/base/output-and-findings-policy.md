# Output and findings policy

Use the skill-specific report format. Findings must be actionable and proportionate: stable identifier when findings apply, location/evidence, consequence, severity or priority where meaningful, confidence, and a smallest safe next step. Distinguish verified findings from questions, assumptions, and clean checks. Preserve source provenance for inherited findings.
