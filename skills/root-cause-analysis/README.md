# root-cause-analysis

This folder contains the canonical prompt for the root cause analysis skill.

## Files

- `SKILL.md`: skill definition and RCA workflow
- `references/`: evidence, timeline, techniques, systemic analysis, prevention, and domains guidance
- `evals/`: incident-style evaluation fixtures

## Summary

Investigate a software failure like a senior engineer running a postmortem and trace the incident back to root causes.

## Compatibility

- Standalone skill folder: yes
- Shared prompt logic: none
- Dependencies on other skills: none
