# debugging

This folder contains the canonical prompt for the debugging skill.

## Files

- `SKILL.md`: skill definition and debugging workflow
- `references/`: evidence, runtime, concurrency, frontend, infrastructure, data, and security guidance
- `evals/`: evaluation fixtures and incident examples

## Summary

Diagnose the real root cause of a bug, crash, failure, or performance problem using evidence-driven investigation.

## Compatibility

- Standalone skill folder: yes
- Shared prompt logic: none
- Dependencies on other skills: none
