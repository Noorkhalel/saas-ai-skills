# refactoring-code

This folder contains the canonical prompt for the refactoring skill.

## Files

- `SKILL.md`: skill definition and refactoring workflow
- `references/`: smells, patterns, architecture, performance, security, testing, and workflow guidance
- `evals/`: evaluation metadata

## Summary

Analyze, plan, and safely execute behavior-preserving refactoring of existing code.

## Compatibility

- Standalone skill folder: yes
- Shared prompt logic: none
- Dependencies on other skills: none
